    Play-Cricket API Documentation¶

    The Play-Cricket API provides read-only access to cricket data held within the Play-Cricket platform — fixtures, results, scorecards, league tables, clubs, teams, and players.

    Who this API is for

    Play-Cricket's API is intended for clubs and competitions to retrieve their own data from the platform for processing elsewhere. It is not available as a service for those looking to build products or applications using cricket data. For more info, read here.

    All endpoints use HTTP GET requests and return JSON. The API is at version v2.

    Base URL 

    https://www.play-cricket.com/api/v2/

    Access required

    All requests require an API token. See Authentication for details on obtaining one.

    API Reference¶

    Organisation¶

    Endpoint Description
    GET /CCBs.json List all County Cricket Boards
    GET /clubs.json List clubs, filtered by county or date
    GET /league_sites.json List all league sites and their URLs

    Teams & Players¶

    Endpoint Description
    GET /sites/{site_id}/teams.json List teams for a club or CCB site
    GET /sites/{site_id}/players List players for a club
    GET /competition_teams.json List teams in a specific division or cup

    Competitions¶

    Endpoint Description
    GET /competitions.json List divisions and cups for a league season

    Matches¶

    Endpoint Description
    GET /matches.json Fixture list — dates, teams, ground, officials. No result data
    GET /result_summary.json Completed matches with result, points, and innings totals
    GET /match_detail.json Full scorecard — batting, bowling, fall of wickets

    Standings¶

    Endpoint Description
    GET /league_table.json Current league table for a division
